ER-модель take-test.online

Логическая структура базы данных PostgreSQL

erDiagram
    users {
        int id PK
        string email UK
        string login UK
        string password_hash
        boolean is_verified
        datetime created_at
    }

    tests {
        int id PK
        string owner FK
        string title
        string type
        json questions
        string share_code UK
        int time_limit
        boolean shuffle_questions
        string folder
        datetime created_at
    }

    submissions {
        int id PK
        int test_id FK
        string name
        int score
        int total
        int answered
        int skipped
        json detailed
        string at
    }

    email_codes {
        int id PK
        string email
        string code
        string purpose
        datetime expires_at
        datetime created_at
    }

    users ||--o{ tests : "creates"
    tests ||--o{ submissions : "has"
    users ||--o{ email_codes : "receives"

Описание таблиц

users

ПолеТипОписание
id PK INTEGER Автоинкремент
email UQ IDX VARCHAR Email пользователя
login UQ IDX VARCHAR Логин (3-30, латиница+цифры+_)
password_hash VARCHAR Хеш bcrypt
is_verified BOOLEAN Подтверждена ли почта
created_at DATETIME Дата регистрации

tests

ПолеТипОписание
id PK INTEGER Автоинкремент
owner FK IDX VARCHAR Логин владельца (users.login)
title VARCHAR Название теста
type VARCHAR Тип: quiz / survey / analytics
questions JSON Массив вопросов
share_code UQ IDX VARCHAR Уникальный код для шеринга
time_limit INTEGER Лимит времени (0 = без лимита)
shuffle_questions BOOLEAN Перемешивать вопросы
folder VARCHAR Название папки
created_at DATETIME Дата создания

submissions

ПолеТипОписание
id PK INTEGER Автоинкремент
test_id FK INTEGER Ссылка на tests.id
name VARCHAR Имя респондента
score INTEGER Набранные баллы
total INTEGER Максимум баллов
answered INTEGER Отвеченных вопросов
skipped INTEGER Пропущенных вопросов
detailed JSON Детальные ответы
at VARCHAR Дата/время прохождения

email_codes

ПолеТипОписание
id PK INTEGER Автоинкремент
email IDX VARCHAR Email получателя
code VARCHAR 6-значный код
purpose VARCHAR register / reset
expires_at DATETIME Срок действия (10 мин)
created_at DATETIME Дата создания
